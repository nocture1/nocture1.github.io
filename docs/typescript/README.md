# TypeScript学習メモ

## 基本
JavaScriptをベースにした静的型付け＋オブジェクト指向言語

## モチベーション
- JavaScriptに型付けを行える
    + 互換性を持たない型に値を割り当てることができない

JavaScriptなら実行時に型の検証が行われるが、事前に行うことで悪い芽を早期に摘める。  
他にもトピックはあるが、まずは的確な型を割り当てることを目標にする。

## 基本編
- 条件分岐に型定義を盛り込むことをゴールにする。

## 初歩編
- 複雑な値を使った条件分岐に型定義を盛り込む。
- 定義した型を再利用する。

## ステップアップ編
- 型の表現によりPrimitive Obsessionから脱却する。
